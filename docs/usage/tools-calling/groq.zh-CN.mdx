# Groq 平台模型 Tools Calling 评测（Llama 3/Mistral）

由于 Groq 本身不支持 stream，因此 Tools Calling 的调用都是以单次请求的方式进行的。

Groq 平台的模型 Tools Calling 能力一览

| 模型         | 支持 Tools Calling | 流式 （Stream） | 并发（Parallel） | 简单指令得分 | 复杂指令 |
| ------------ | ------------------ | --------------- | ---------------- | ------------ | -------- |
| LLAMA3 70B   | ✅                 | ❌              | ✅               | 🌟🌟         | 🌟🌟     |
| Mixtral-8x7B | ❌                 | ❌              | ❌               | ⛔           | ⛔       |
